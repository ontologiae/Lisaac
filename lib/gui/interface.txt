| Interface automatique |
*-----------------------*
Définition
On a deux éléments génériques:
1-ITEM:
Elément de base d'une entité interface ayant toujours un titre sémantique
virtuel et une représentation variable:
bouton, saisi, check, ...
Exemple de titre virtuelle:
"save" à traduire concrêtement selon le contexte par :
- un item dans un menu avec "save" ou "sauvegarder" selon la langue...
- un bouton texte "save" ou "sauvegarder"
- un bouton icone avec une disquette

2-GROUP:
Ensemble d'ITEM pouvant prendre différente forme:
Window, menu, groupe dans une fenetre, ...
(un GROUP est aussi un ITEM)

Avec ca, nous avons un arbre de GROUP (branche) et ITEM (feuille)
d'interface. Selon les possibilité (taille d'affichage, input clavier,
mouse, pen, ...) l'arbre prend une forme ou une autre... en créant des
object de l'interface concret ...
