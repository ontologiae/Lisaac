---------------------------------------
-- LiA: Lisaac Android - APK Builder --
---------------------------------------
          By Sonntag & Cavallero, 2018.

Install:
========

- Java v8 (minimum)
- sudo cp android/* /usr/lib/.

Warning: 
========

- Android plateform #16 (minimum)

Configuration:
==============

- Active Parameter/debug mode USB 
- Connect your phone/tablet on your computer.

Test:
=====

- cd example/test2d
- For Linux test version:

  lisaac test2d -boost

- For Android test version:

  lisaac test2d -target android -boost

Good luck !
